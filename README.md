## Workshop de Portfólio - CP2 Júnior

### 📌 Sobre o Projeto
Este repositório contém o código desenvolvido durante o curso/workshop de criação de portfólio ministrado pela **CP2 Júnior**. 

### 🚀 Tecnologias Utilizadas
* **HTML5:** Para a estruturação do conteúdo.
* **CSS3:** Para a estilização e layout da página.

### 📚 Objetivo
O foco deste repositório foi colocar em prática os conceitos básicos de desenvolvimento web front-end ensinados durante o workshop da CP2 Júnior, focando em como estruturar e estilizar uma página do zero.
